Form Practice

Below is the collection of three simple forms. These forms are hooked up to some JavaScript that powers the functionality of the forms. These examples scratch the surface of what forms can do and serve as an introduction to the very basic form layout, styling, and parts

Parts of the forms contain id and name attributes. "id" attributes are used to identify specific elements on a page and there should only be one uniquely names id on a given page. For example, if we had an id of "fname" we would only use that once on the page. The name attribute is often used for identifying elements in a form. We will try to keep these names unique as well
