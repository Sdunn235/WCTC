Debugging with Developer Tools

We are going to use debugging tools to locate specific items in are markup and help inform our decisions on what to fix.

Downloading Chrome

Each browser has its own set of debugging tools. Chrome's debugging tools have been an industry standard for many years and we will focus on Chrome's debugging tools for this class. You should familiarize yourself with other debugging tools as well. Make sure to download Chrome.

Inspector Tool

The Inspector Tool for Debugging

Using the inspector tool, figure out why the following issues are occuring and fix them:

  • Why does the navigation not start on the left margin?
  • Why does the list not have the dark color applied?
  • Why do the strong and em elements in the header not display on their own lines?
  • Apply the class of highlight to the “Download Chrome” Section. Why did the underline stay the same color instead of going to the light variable like the rest of the content?
