Styling Additions and Updates

This website consists of the basic framework that we have set up in the previous labs. This is a jumping off point and the styles are nowhere near perfect. Frameworks serve as a starting point for our websites. When using frameworks, there generally always needs to be some element of customization and personalization as frameworks are often the baseline. We will do a few edits here, but understand that this is still not a complete solution.

Change your colors

Use Coolors to pick out a new set of colors. Though we use a tool like Coolors, it is not perfect and practicing picking a good palette is a skill we continually work on.

Coolors.co

Change the font

Update the fonts on the website. In the styles.css page, you will see that there is a variable for header-typeface and a variable for body-typeface. Update both values to use a Google font. Make sure to load the fonts in your head section on each page.

Google Fonts

Creating a Favorite Icon

Use a favicon generator to add a favicon to the root of your website. This is the icon that shows up in the tab at the top of the page and when you favorite a page. You may need to hard refresh by doing ctrl + f5 or cmd + f5. If you have functions enabled, you will most likely need to hold your fn key as well.

favicon.io

Add a small button option

Add another button option called btn--sm which will have a font-size of 1em and padding of 10px. This class will be used in conjunction with the btn class and should override the font-size and padding of the btn class. Apply that button to the elements with the btn classes on this page
