Basic jQuery

Use the jQuery function with CSS selectors as the argument (string) in order to select elements as a jQuery object. Try some basic selections. Remember that for jQuery to work on a page, two things need to be included: a link to the jQuery library (CDN script) and the document ready function.

id Selection

jQuery uses CSS selectors. The octothorpe (number sign / pound sign / hash tag) is used for selecting ids.
