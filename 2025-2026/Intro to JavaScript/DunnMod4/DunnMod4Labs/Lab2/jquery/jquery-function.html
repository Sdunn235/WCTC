jQuery Function

The jQuery Function returns a collection of matched elements either found in the DOM based on passed argument(s) or created by passing an HTML string. Look at the overloads for the jQuery function. It will act differently based on the arguments.

jQuery Function

jQuery()

jQuery Function Shorthand

$()

Example

The following selects every element with the class of active and return a jQuery object for the selection:

$('.active')

Look at the console window for the logged result. Notice anything interesting?

.practice-class
.practice-class.active
.practice-class.active
.practice-class

jQuery Object

Make special note that the type of object returned is not a NodeList, but a jQuery Object. As such, the jQuery object has access to the members of jQuery objects.
