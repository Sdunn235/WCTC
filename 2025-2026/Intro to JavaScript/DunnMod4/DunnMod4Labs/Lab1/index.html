Events

This lab examines using events in JavaScript utilizing the addEventListener() method. Event listeners are used to wait for actions to occur in the HTML and then "handle" the functionality once the event is triggered.

Nodes

A node is an abstraction of objects like Document and Element. For our purposes, we will often use Element and Node interchangeably. An Element Object is a type of Node.
