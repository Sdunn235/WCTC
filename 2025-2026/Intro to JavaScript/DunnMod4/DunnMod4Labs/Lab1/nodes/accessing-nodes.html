Accessing Nodes

Nodes

A node is an abstraction of objects like Document and Element. For our purposes, we will often use Element and Node interchangeably. An Element Object is a type of Node.

There are several different ways to access nodes. Here are three ways to do it, but there are more than this.

Mozilla: Nodes

w3s: Nodes

getElementById('id')

getElementById() is a method that returns a HTMLElementObject of the first element that matches the argument for id.

querySelector('cssSelector')

querySelector() returns an Element Object for the first matching element based on the CSS selector argument. Note that CSS selectors are what are used in the CSS file. Ids are prefixed with # and classes are prefixed with .

Log the div.log-me to the console window. Note which Element Object is returned.

querySelectorAll('cssSelector')

querySelectorAll() returns a static/non-live NodeList Object for the all matching elements based on the CSS selector argument. Static or non-live means that if additional elements are added that match the selector after the method has been called, the NodeList will not include those new elements.
