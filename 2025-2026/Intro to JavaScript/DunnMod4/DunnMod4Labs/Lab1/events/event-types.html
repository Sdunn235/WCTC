Event Types

The Event Object

Event Types are the type of event that was triggered. click, change, and submit, are common types of events.

w3s: Event Object Type

Load Event

Attach an event to the window object using an event listener. Listen for the event of 'load'. Log the timeStamp of the page loaded event to the console.

Mozilla: Load Event

Focus and Blur

Add two events to the following input#focus-and-blur. One event should trigger on focus, and one event should trigger on blur. On focus, give the input the class of 'active'. You can use the keyword this and the classList to acomplish this.

w3s: Focus Event

w3s: Blur Event

Mouse Position

Add a mousemove event to the document. Upon triggering, the event listener should update input#mouse-xy to display the x and y coordinates in this format: (x, y). Adding an event to the document object is like adding an event to the window object.

w3s: Mouse Move Event
