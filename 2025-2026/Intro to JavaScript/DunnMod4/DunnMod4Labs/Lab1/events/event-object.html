The Event Object

The Event Object

When an event occurs, the event object gives information about the event that occurred. When a function is called for an event (named or anonymous), the first parameter of the function is the event object. This event object does not need to be added to the parameters, but will be the first parameter that gets added. The parameter can have any name, but it is typically called "e".
