Extra: Family Methods

Traversing the DOM

Completion of this lab is not required and is extra credit. Using either JavaScript or jQuery, make it so clicking a button moves the class of active from the current element to the appropriate element based on the button selection.

If you try to access an element that does not exist, the program may error. Validation for this is not required for credit.

  • Next Sibling
