Common  jQuery Methods

HTML Content Method

.html() Method

Documentation

Get the HTML contents of the first element in the set of matched elements or set the HTML contents of every matched element.

.html() to get information

Use the .html() method with no arguments to get the contents of the element with the id of "html-as-get".

Log the results to the console

#html-as-get

Log the results of the HTML

.html() to set information

Use the .html() method with HTML String as the argument to update the contents of the element with the id of "html-as-set" to the following:

<h4>This has been updated</h4>
<p>with an HTML string</p>

#html-as-set

Replace the content in this div

.html() to set information on multiple elements

Use the .html() method with a HTML String as the argument to update the contents of all elements with the class of "html-set-many" to the following: 

<h4>One method</h4>
<p>updates all of these</p>

.html-set-many

One method, many elements

.html-set-many

One method, many elements

.html-set-many

One method, many elements

.html-set-many

One method, many elements

Text Content Method

.text() Method

Documentation

Get the combined text contents of each element in the set of matched elements, including their descendants, or set the text contents of the matched elements.

.text() to get information

Use the .text() method to output the text of the div with the id of "text-as-get"

Log the results to the console

#text-as-get

Log the results of the HTML

.text() to set information

Use the .text() method to update the text of the div with the id of "text-as-set" to the following:

This text has been updated

#text-as-set

Update the text on this div

.text() to append information

Use the .text() method get the text of the p in the div with the id of "text-to-append" then concatenate your name to the end and replace the p with your appended text.

#text-to-append

Append your name here:

.text() to append a list

Use the .text() method to append " easily updated" to each list item in the div with the id of "text-in-a-list" using only a single selector.

#text-in-a-list

  • Item Two

  • Item Four

Working with Values

.val() Method

Documentation

Get the current value of the first element in the set of matched elements.

.val() to get values

Use the .val() method get the values of the div with the id of the following:

  • get-val-text-input
  • get-val-number-input
  • get-val-select

Log the values and the data types. Use the typeof operator

.val() to set a value

Use the .val() method to set the value of the input with the id of "set-val-text-input" by using an argument of "Updated"

Manipulating Styling

.css() Method

Documentation

Get the value of a computed style property for the first element in the set of matched elements or set one or more CSS properties for every matched element.

.css() to get values

Use the .css() method get the background color of the div with the id of "get-bg-css" by using the correct property as the argument.

#get-bg-css

Get the background color

.css() to set values

Use the .css() method set the background color of the div with the id of "set-bg-css" by providing the first argument of the property, and the second argument of the new value.

#set-bg-css

Set the background color to #D3D3D3

.addClass() Method

Documentation

Adds the specified class(es) to each element in the set of matched elements.

.addClass() to add a class

Use the .addClass() method to add the class of "active" to the div with the id of "add-active"

#add-active

Add the class of active to this div

.removeClass() Method

Documentation

Remove a single class, multiple classes, or all classes from each element in the set of matched elements.

.removeClass() to remove a class

Use the .removeClass() method to remove the class of "off" to the div with the id of "remove-off"

#remove-off

Remove the class of off from this div
