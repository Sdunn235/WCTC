jQuery

jQuery is a JavaScript library that is meant to simplify much of what we do. To use jQuery, you need to include the library in your site, either by having all of the library files on your site, or linking to an online CDN (Content Delivery Network) that hosts the library files. Google hosts jQuery and you will typically see this script added to link to the Google jQuery files.


jQuery Setup

Include the following script tag in your html file:

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script>

Wrap your code in a special function called the Document Ready Function. Additional code will go inside this function:

$(document).ready(function(){
    /* jQuery and JavaScript code here */              
});


Using jQuery

You can use the jQuery library. You can select elements using the jQuery selector syntax $() in conjunction with css selectors. This would select the button with the id of jQueryExampleButton: 

$('#jQueryExampleButton');

And this adds an event listener to that button via jQuery that runs an anonymous function:

$('#jQueryExampleButton').click(function(){alert("Using jQuery")});
