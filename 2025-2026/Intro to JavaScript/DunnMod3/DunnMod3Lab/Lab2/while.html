While Loops

While loop that continues with user confirmation

This while loop is a pre-test loop. The test expression will ask for user confirmation and then run the loop. The code block inside the while loop may never run if the user selects cancel immediately.

While loop that counts to a value

You can use a while loop to iterate a certain number of times, much like a for loop. In fact, a while loop is the basis of a for loop.
